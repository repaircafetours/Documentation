Architecture of the Repair Café Tours application as follows

- Web application for administration
- Wordpress plugin for managing visitors
- HumHub plugin for volunteers **(not implemented yet)**
- Backend API for supporting all of this app

You can find the source code of this project on [GitHub](https://github.com/repaircafetours).

For backend development, we use [Laravel](https://laravel.com/).

For frontend development, we use [React](https://reactjs.org/) and [Next.js](https://nextjs.org/).

You can find full documentation for the backend API on [Backend](./backend/intro) documentation.

You also can find full documentation for the frontend application on [Frontend](./frontend/intro) documentation (but is not written yet).
